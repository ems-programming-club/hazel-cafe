# ☕ Hazelnut Café — Website

A multi-page website for Hazelnut Café & Bakery, a family-run café in Fleetwood, Surrey. Built to answer customer questions automatically so the owner can focus on baking.

---

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage / dashboard — hero feature, drink & bakery previews, FAQ highlights, contact info |
| `featured.html` | Featured items — showcase cards for bestsellers with photos and descriptions |
| `menu.html` | Full menu — all drinks, bakery items, light food, custom cakes, and dietary info |
| `faq.html` | FAQ page — 20 expandable Q&A items covering everything customers ask |
| `style.css` | Shared stylesheet for all pages |

---

## Features

- **Pill navigation bar** — consistent nav across all pages with an active-state highlight
- **Hero panel** — warm gradient banner on the homepage featuring the daily special (cinnamon buns)
- **Menu previews** — drink and bakery sidebars on the homepage with live pricing
- **Featured showcase** — photo cards for the top three bestsellers (cinnamon bun, hazelnut latte, butter croissant)
- **Full menu page** — organized into Coffee & Drinks, Bakery, Light Food, Custom Cakes, and Dietary Info sections
- **Custom cake info** — size/price grid plus order rules (72-hour notice, $25 deposit, flavour options, 6-word message limit)
- **Dietary section** — gluten-free, vegan, and allergy information clearly laid out
- **Expandable FAQ** — 20 accordion-style Q&A items using `<details>` / `<summary>` — no JavaScript required
- **FAQ preview on homepage** — four most common questions surfaced right on the homepage
- **Contact section** — location, hours, phone, social, payment methods, and seating info in a grid layout
- **AI chat widget** — Chatbase-powered assistant embedded on every page for instant customer Q&A
- **Responsive design** — mobile-friendly layouts via CSS Grid breakpoints at 700px and 1000px
- **Image-text effect** — coffee photo clipped into the homepage headline text

## Problem Solved

Before this website, Hazelnut had **no online presence except a stale Instagram page**. Anyone searching "bakery near me" or "café in Fleetwood" found nothing useful — no menu, no hours, no prices — and walked into the chain café down the street instead.

This site fixes that directly:

- **Hours and location are front and center** — a new customer can confirm they're open before leaving the house
- **Full pricing is public** — no need to call just to ask how much a latte costs
- **Dietary info is clear and specific** — gluten-free and allergy customers get a trustworthy answer without having to phone in
- **Custom cake rules are spelled out** — cake customers know exactly what to expect before they reach out
- **The AI chat widget catches anything the pages don't** — open questions get answered instantly, 24/7

The result: a new customer searching for a local café now finds everything they need to walk through the door.
